# Factorio Prints

[![Join the community on Discord](https://img.shields.io/badge/Join%20the%20community-blueviolet.svg?style=for-the-badge&logo=Discord&labelColor=000000&logoWidth=20)](https://discord.gg/uvUUw5a9Qc)

A community site to share and browse blueprints for the game Factorio.

## Features

- Share your Factorio blueprints with the community
- Browse blueprints by tags
- Search by recipe, entity, game version, and game mod
- Favorite blueprints for easy access

## Contributing

Feel free to open issues and pull requests. Join our Discord community to discuss features and get help.
